Resources and more

A collection of useful links and websites to help further you web development and programming skills

  • Codecademy
    A great website which has interactive tutorials where you can learn a variety of programming languages and skills right from the browser.
  • W3Schools
    A collection of tutorial, references and tips provided by W3 for all things web related.
  • Bootstrap Components
    A handy reference of over a dozen reusable components built to provide iconography, dropdowns, input groups, navigation, alerts, and much more provided by Bootstrap.
  • Bootstrap CSS Reference
    A reference to Bootstrap's global CSS settings, fundamental HTML elements styled and enhanced with extensible classes, and advanced grid system.
  • HTML Reference
    The Mozilla Developer Network's list of every HTML element and information about them.
  • CSS Reference
    The Mozilla Developer Network's list of all CSS properties and values along with additional information and usages.
  • Foundation
    Another framework for building responsive websites similar to Bootstrap. Make sure to have a look and experiment with it and also check out the many other frameworks that exist.
  • Code School
    Another website which teaches many different of different programming languages and covers a range of web development techniques from within the web browser with a mix of free and subscription based courses.
