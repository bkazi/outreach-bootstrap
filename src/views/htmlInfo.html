Introduction to HTML and CSS

An introductory guide to creating amazing webpages using Bootstrap 3

HTML

HTML is made up of HTML elements, which live inside angled brackets. These are usually made up of two tags: an opening and a closing tag (which has an extra forward slash in it). Each HTML element tells the browser something about the text that sits between those opening and closing tags.

Sometimes the opening tag of the element contains more information, in something called an attribute. These are made up of two parts: a name and a value, separated by an equals sign.

Structure of a Website

<body>
Everything inside this element is shown inside the main browser window.

<head>
Before the <body> element you will often see a <head> element. This contains information about the page (rather than information that is shown within the main part of the browser window.

<title>
The contents of the <title> element are either shown in the top of the browser on the tab for that page. 



		
<html>
<head>
<title>This is the title of the Page</title>
</head>
<body>
<h1>This is the Body of the Page</h1>
<p>
Anything displayed within the body of the web page is displayed within the main browser window.
</p>
</body>
</html>

Lists .. Lists .. Lists

There are 2 kinds of list in HTML, ordered and unordered lists. See what the code below does to the webpage:

		
<html>
<body>
<ul>
<li>1kg King Edward potatoes</li>
<li>100ml milk</li>
<li>50g salted butter</li>
<li>Freshly grated nutmeg</li>
<li>Salt and pepper to taste</li>
</ul>
<ol>
<li>Chop potatoes into quarters</li>
<li>Simmer in salted water for 15-20 minutes until tender</li>
<li>Heat milk, butter and nutmeg</li>
<li>Drain potatoes and mash</li>
<li>Mix in the milk mixture</li>
</ol>
</body>
</html>

<ul>
The unordered list is created with the <ul> element

<ol>
The ordered list is created with the <ol> element

<li>
Each item in the list is placed between an opening <li> tag and a closing </li> tag. Browsers indent lists by default



Links!

		
<a href="http://www.google.com">Google!</a>

Links are created using the <a> element. Users can click on anything between the opening <a> tag and closing </a> tag. You can specify which page you want to link to using the href attribute.

Images

		
<img src="img/image.jpg" height="300" width="400" alt="image">

<img>
To add an image into the page you need to use an <img> element. This is an empty element (which means there is no closing tag).

Attributes:

src - This tells the browser where it can find the image file
height - This specifies the height of the image in pixels
width - This specifies the width of the image in pixels
